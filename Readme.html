Welcome to CricketDarts.net.

This is a super sleek and intuitive online scoreboard that makes it easy for up to 4 players to track their cricket darts matches in real-time. It’s a great way for players to focus on the game rather than the math!

		How to use site

	Large iPads work best. 
	Remember to set screen not to sleep.
      
	1. Select # of Players.

	2. Pick a Color for each column.

	3. Enter Name for each column.
		
		For Teams enter two names per column.

		Helpful note:
		Tap to the right of existing text, 
		so backspace can be used to wipe out P1 
	
	- Tap cell to add one mark at a time.

	- 3 marks opens target for player. 
	once open, Player can score until 
	opponent closes target with 3 marks.

	- Hold cell to undo mark.
	Remain Still while holding. 
	Very sensitive.

	- First Player/Team to 3 mark each number
	and have high score wins.

	- Refresh Page to Start Over.
